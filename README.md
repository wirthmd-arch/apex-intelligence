Apex Intelligence — AI-Powered Investing Dashboard
A fully offline, browser-based investing dashboard with 60 tabs covering portfolio tracking, live market data, AI-powered analysis, research tools, trading calculators, and financial planning. No server, no subscription, no installation — just open the HTML file in your browser.

What's Inside
Portfolio — Positions, Watchlist, Alerts, Top Grades, Dividend Tracker, Portfolio Analytics, Tax Lots, Paper Trading, Net Worth, Stress Test, Benchmark Comparison, Rebalancing, Crypto Portfolio
Markets — Macro Dashboard, Economic Calendar, Earnings Calendar, Rates & Mortgage Calculator, Crypto, Funds & ETFs, Bonds, News Sentiment, Economic Indicators, Market Heatmap, Sector Rotation, Market Breadth, IPO Calendar, Stock Splits
Research — Stock Screener, Stock Comparison, Fundamentals, ETF X-Ray, Congress Tracker, Wall St Firms, Analyst Ratings, Morningstar-Style Ratings, Valuation & DCF, Short Flow
Trading — Fast Money, Options Chain, Technical Analysis, Stock Advisor, Alert Triggers, Position Sizing, DCA Simulator, Trade Journal, Options Risk Graph, Options Greeks
Planning — Broker Guide, Inflation Calculator, Social Security Optimizer, Credit & Debt Tracker, Life Insurance Estimator, Retirement Planner
More — Event Plays, Stock Picks, Trade Advisor (AI), Settings

How to Use

Click the green Code button above → Download ZIP
Extract the ZIP anywhere on your computer — all 64 files go in the same folder
Open apex_intelligence.html in Chrome or Edge
Navigate using the tab bar — each section also opens as its own standalone page


Important: Keep all files in the same folder. The app will not work if you move files separately.


Live Data
Most tabs attempt to fetch live data automatically on load via Yahoo Finance. Where live data is unavailable, seed data is shown with a yellow "Seed Data" badge and a "Refresh with AI" button.
Live data includes: S&P 500, Nasdaq, VIX, Gold, 10-Year Yield, Dollar Index, Bitcoin, sector ETF prices, top mover prices, and individual stock quotes.

AI Features (Optional)
To unlock AI-powered features — Daily Market Briefing, Stock Advisor, Trade Analysis, Sector Rotation AI, Economic Indicators Refresh, and more:

Go to More → Settings
Enter your OpenAI or Google Gemini API key
Keys are stored only in your browser's local storage — never transmitted anywhere except directly to the AI provider you choose

No AI key is required to use the rest of the dashboard.

Privacy

All your data (positions, watchlist, trades, notes) is stored locally in your browser via localStorage
Nothing is sent to any server — there is no backend
Live price fetches go directly from your browser to Yahoo Finance
AI queries go directly from your browser to OpenAI or Google — no intermediary


Requirements

Chrome or Edge recommended (required for live price fetching from file://)
No internet connection required for core features
No Node.js, no npm, no build step
